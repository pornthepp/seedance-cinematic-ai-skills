# Short Film Artistic Narrative Analysis

Help a short film become sharper, more filmable, and more emotionally
concentrated.

## Token Discipline

Start here. Read one reference only when needed:

| Need                                                                                 | Read                                      |
| ------------------------------------------------------------------------------------ | ----------------------------------------- |
| Full critique, rewrite plan, pitch note, festival-facing analysis                    | `references/short-film-analysis-guide.md` |
| Full worked examples: script critique, festival feedback, diagnosing pacing          | `references/examples.md`                  |
| Advanced: pressure cooker model, ending taxonomy, festival heuristics, scene economy | `references/advanced-techniques.md`       |

## Quick Workflow

1. Identify the central wound, question, or contradiction.
2. Compress the story to one visible transformation.
3. Check whether each scene creates pressure, choice, or consequence.
4. Convert abstract themes into filmable actions, images, objects, rhythm, and
   sound.
5. Review social context and symbolism without flattening characters into
   messages.
6. Strengthen the ending so it lands as image, action, or irreversible emotional
   shift.
7. Output notes in clear priorities: keep, cut, clarify, intensify.

## Output Shape

- Core dramatic engine
- What is working
- Main risk
- Structural notes
- Character notes
- Visual/sound notes
- Concrete rewrite actions

## Quality Gate

A short film should not feel like a compressed feature. It should feel like one
precise cinematic event with emotional aftershock.
