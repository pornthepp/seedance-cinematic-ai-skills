# Seedance Cinematic AI Skills

A collection of specialist skills for cinematic creative work. Each skill is a focused workflow — load only the one you need.

## Starting Point

Read `skills/SKILL_INDEX.md` to find the right skill for the task. If the request is vague, read `skills/PREFLIGHT_ROUTER.md` first.

## Skills

| Skill | When to use |
| ----- | ----------- |
| `skills/cinematic-camera-composition-director/GUIDE.md` | Camera angles, shot sizes, lens choices, storyboards, composition |
| `skills/cinematic-color-look-development/GUIDE.md` | Color palettes, look development, LUT planning, color grading |
| `skills/cinematic-music-soundscape-design/GUIDE.md` | Film music, sound design, cue planning, silence, ambience |
| `skills/ethical-character-representation/GUIDE.md` | Character representation, cultural signals, bias, identity systems |
| `skills/gpt-image-2-photoreal-synthesis/GUIDE.md` | Text prompts for GPT Image 2 — product, portrait, editorial, typography |
| `skills/modern-acting-performance-mastery/GUIDE.md` | Acting direction, emotional beats, dialogue delivery, body and voice |
| `skills/seedance-cinematic-prompt-director/GUIDE.md` | Seedance 2.0 video prompts — action, storyboard, anti-slop refinement |
| `skills/short-film-artistic-narrative-analysis/GUIDE.md` | Short film critique, narrative structure, festival prep |

## Rule

Load one skill at a time. Do not load all skills for a cinematic request.
