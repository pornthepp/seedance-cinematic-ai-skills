# Cinematic Music Soundscape Design

Design sound from story function, not from generic mood words.

## Token Discipline

Start here. Read one reference only when needed:

| Need                                                                                 | Read                                |
| ------------------------------------------------------------------------------------ | ----------------------------------- |
| Cue sheets, sound palette detail, motif planning, structured sound review            | `references/soundscape-guide.md`    |
| Full worked examples: tension scene, score direction, sound palette for a film       | `references/examples.md`            |
| Advanced: frequency map, silence types, spotting mistakes, ambient layers, sound POV | `references/advanced-techniques.md` |

## Quick Workflow

1. Read the scene before scoring it: action, emotion, subtext, point of view.
2. Decide whether sound should support, contrast, mislead, reveal, or withhold.
3. Choose the lead element: music, ambience, silence, foley, voice, or sound
   design.
4. Define rhythm, density, texture, register, and space.
5. Spot cue start and exit points.
6. Check whether silence would be stronger than music.
7. Output a cue note, sound palette, or review table.

## Output Shape

- Emotional sound intention
- Lead audio element
- Palette or instrumentation
- Rhythm and movement
- Spatial placement
- Cue start and exit
- Mix note

## Quality Gate

Sound must change what the audience understands or feels. Avoid wallpaper
scoring, constant music, and cues that repeat what the image already says.
