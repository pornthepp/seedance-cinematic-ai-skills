# ชุดสกิลงานภาพยนตร์

ภาษา: [English](README.md) | **ไทย**

ชุดสกิลที่ช่วยให้ AI agent ทำงานสร้างสรรค์ด้านภาพยนตร์ได้ดีขึ้น — ตั้งแต่เขียน prompt รูปภาพ กำกับการแสดง ออกแบบสี วางแผนเสียง ไปจนถึงวิเคราะห์หนังสั้น แต่ละสกิลเป็นแนวทางการทำงาน (Workflow) สั้นๆ ที่ agent จะโหลดเข้ามาทำงานโดยอัตโนมัติเฉพาะตอนที่ต้องใช้

## มีอะไรบ้าง

**9 สกิล** ครอบคลุมงานสร้างสรรค์ด้านหนังหลักๆ:

| สกิล                                     | ช่วยเรื่อง                                                                                                                     |
| ---------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------ |
| `generative-ai-visual-prompt-director`   | เปลี่ยนบรีฟงานภาพให้เป็น prompt ภาพหรือวิดีโอที่เป็นกลาง ปรับแนวคิดข้ามแพลตฟอร์ม และวิเคราะห์แก้ไขภาพที่ยังไม่สมบูรณ์           |
| `cinematic-camera-composition-director`  | มุมกล้อง ขนาดช็อต เลือกเลนส์ สตอรี่บอร์ด จัดเฟรม                                                                               |
| `gpt-image-2-photoreal-synthesis`        | เขียน prompt ให้ GPT Image 2 สร้างภาพสมจริง — สินค้า พอร์ตเทรต ตัวอักษร                                                        |
| `cinematic-color-look-development`       | จานสี ออกแบบลุค ตรรกะแสง/สี โน้ตเกรดสี                                                                                         |
| `cinematic-music-soundscape-design`      | กำกับดนตรี เอฟเฟกต์เสียง ความเงียบ วางแผนบรรยากาศเสียง                                                                         |
| `short-film-artistic-narrative-analysis` | วิจารณ์บทหนังสั้น โครงสร้างเรื่อง เตรียมส่งเทศกาล                                                                              |
| `seedance-cinematic-prompt-director`     | เขียน prompt วิดีโอสำหรับ Seedance 2.0 — คลิปแอ็กชัน สตอรี่บอร์ด โฆษณา มิวสิควิดีโอ งานการศึกษา พร้อมระบบคัดกรองคำอวยอัตโนมัติ |
| `modern-acting-performance-mastery`      | กำกับนักแสดง จังหวะอารมณ์ พูดบท ภาษากาย                                                                                        |
| `ethical-character-representation`       | เช็คตัวละครว่ามี stereotype, อคติ, หรือปัญหาการนำเสนอไหม                                                                       |

## ทำงานยังไง

คลังเก็บข้อมูลนี้ใช้ระบบ **ค้นพบความสามารถอัตโนมัติ (Customizations Skills Auto-Discovery)** ของ Gemini:
1. ทุกสกิลจะถูกวางอยู่ในโฟลเดอร์ของตัวเองที่ระดับนอกสุด (Project Root)
2. ในแต่ละโฟลเดอร์จะมีไฟล์ชื่อ `SKILL.md` ที่เก็บรายละเอียดหัวข้อความสามารถ (`name` และ `description`)
3. เมื่อ AI agent ได้รับมอบหมายงาน ระบบจะสแกนโฟลเดอร์ระดับนอกสุดและจับคู่ความต้องการกับคำอธิบายของแต่ละความสามารถ และโหลดคำแนะนำใน `SKILL.md` ที่เกี่ยวข้องเข้ามาใช้งานโดยอัตโนมัติ
4. หากงานต้องการข้อมูลเชิงลึกเพิ่มเติม agent จะเข้าไปอ่านเฉพาะไฟล์อ้างอิงภายในโฟลเดอร์ความสามารถนั้นๆ เพื่อหลีกเลี่ยงการโหลดข้อมูลมากเกินไปจนทำให้ระบบมึนงง

## โครงสร้างในแต่ละโฟลเดอร์สกิล

ทุกสกิลมีโครงสร้างเหมือนกันดังนี้:

```text
[ชื่อโฟลเดอร์สกิล]/
  SKILL.md            ← เริ่มที่นี่ workflow สั้นๆ + เช็คคุณภาพ
  templates/
    output-template.md  ← แม่แบบผลลัพธ์เปล่าสำหรับนำไปใช้งาน
  references/
    core-guide.md       ← คำศัพท์ เช็คลิสต์ ตารางรีวิว
    examples.md         ← ตัวอย่างจริง input → output เต็มๆ
    model-quirks.md     ← เคล็ดลับเฉพาะรุ่น (GPT Image 2 / Seedance เท่านั้น)
    advanced-techniques.md  ← เทคนิคขั้นสูงระดับมืออาชีพ
```

## เริ่มใช้ยังไง

**ถ้าคุณเป็น AI agent:** เนื่องจากแต่ละความสามารถถูกย้ายมาอยู่ที่โฟลเดอร์หลักแล้ว ระบบจะทำการโหลดไฟล์ `SKILL.md` ที่เกี่ยวข้องให้คุณโดยอัตโนมัติเองตามประเภทงาน ให้ปฏิบัติตามขั้นตอนนั้นๆ และเปิดอ่านไฟล์อ้างอิงในโฟลเดอร์เมื่อจำเป็นต้องใช้ข้อมูลเชิงลึกเท่านั้น

**ถ้าคุณเป็นคน:** เปิด `[ชื่อโฟลเดอร์สกิล]/SKILL.md` ของความสามารถที่สนใจเพื่อใช้เป็นเช็คลิสต์กำกับงานแบบเร็วๆ ได้เลย หากต้องการดูตัวอย่างจริงสามารถไปดูได้ที่ `references/examples.md` ของแต่ละโฟลเดอร์

## ติดตั้ง

คัดลอกโฟลเดอร์ความสามารถที่ต้องการไปไว้ใน skill directory ของ AI agent หรือใช้ทั้งโฟลเดอร์นี้เป็นคลังความสามารถหลัก

## ทำไมถึงออกแบบแบบนี้

- **รองรับระบบสแกนอัตโนมัติ**: เราได้เปลี่ยนมาใช้การค้นหาอัตโนมัติผ่านไฟล์ `SKILL.md` ที่อยู่ในโฟลเดอร์นอกสุดของแต่ละความสามารถ ทำให้การเรียกใช้มีความแม่นยำและรวดเร็วกว่าเดิมมาก
- **กฎการใช้ภาพอ้างอิงเหนือข้อความ (Ref-over-Text Rule)**: กำหนดมาตรฐานสำหรับการทำงานกับภาพอ้างอิง (เช่น Identity, Style, Composition) โดยให้คุณลักษณะของรูปภาพเป็นตัวชี้นำหลัก และให้เขียนข้อความเฉพาะเพื่อระบุจุดที่เปลี่ยนแปลงหรือรายละเอียดเพิ่มเติมเท่านั้น เพื่อหลีกเลี่ยงข้อความที่ซ้ำซ้อนหรือขัดแย้ง ซึ่งมักจะส่งผลให้ภาพที่เจนออกมาเกิดความผิดเพี้ยน
- **สกิล GPT Image 2** มีไว้เขียน prompt ข้อความอย่างเดียว — ไม่มีโค้ด API, ไม่ปนคำสั่งของ Stable Diffusion หรือ Midjourney เพราะเครื่องมือพวกนั้นมีภาษาเฉพาะของตัวเอง
- **สกิล Seedance** ออกแบบเฉพาะสำหรับ Seedance 2.0 — บอกความยาวและขนาดความเคลื่อนไหวที่เหมาะสม พร้อมระบบ "สั่งฉาก ไม่ใช่ตกแต่งคำ" ที่ห้ามใช้คำชมลอยๆ เช่น "cinematic" แต่บังคับให้กำกับทิศทางกล้องและฉากจริงแทน
- **แม่แบบฉากของ Seedance** ครอบคลุมตั้งแต่งานแอ็กชัน บทพูด โฆษณา มิวสิควิดีโอ ไปจนถึงสื่อการเรียนรู้
- **สกิลตัวละคร** ไม่ได้แค่บอกกว้างๆ แต่มีเกณฑ์ให้คะแนนความเหมาะสม 1-5 และมีเช็คลิสต์เฉพาะแนวหนัง (แอ็กชัน, สยองขวัญ, รัก, ไซไฟ, ดราม่า)
- **ตัวอย่างจริง** ทุกสกิลมีตัวอย่างชัดเจนแบบก่อนทำ-หลังทำ เพื่อช่วยในการศึกษา
- **เคล็ดลับเฉพาะรุ่น** มีการรวบรวมพฤติกรรมเฉพาะตัวของปัญญาประดิษฐ์แต่ละรุ่นที่หาอ่านไม่ได้ตามคู่มือทั่วไป
