# Filter-Safe Vocabulary

Use professional, non-graphic, production-context language. This reference
helps rewrite prompts that trigger content filters while preserving
creative intent. Do not use this to bypass safety systems.

## Surface Wording Replacements

| Risky surface wording | Safer production wording |
|---|---|
| violent impact | high-energy collision, non-graphic action beat |
| weapon close-up | prop object held safely, action-scene staging |
| blood | red fabric accent, colored liquid, non-graphic aftermath |
| injury | visible fatigue, dramatic tension, character distress |
| fight | choreographed action sequence, staged confrontation |
| celebrity face | original character with broad archetype traits |
| brand logo | generic product mark or blank label |
| named franchise | original world with similar genre function |
| explosion | high-energy burst, debris scatter, shockwave |
| gun / firearm | action prop, tactical equipment (context-dependent) |

Preserve intent. Change risky surface wording.

## False-Positive Reduction

Use this only for safe prompts that are being misunderstood by filters.
The repair is clarity, not evasion.

| Ambiguous wording | Safer clarification |
|---|---|
| dark fight scene | staged confrontation, choreographed action beat, no graphic injury |
| knife on table | safe prop object on a table, not used for harm |
| intense hostage mood | locked-room tension, psychological pressure, no violence shown |
| character bleeding | character showing visible exhaustion and distress |
| war zone | post-conflict environment, rubble and dust, no active violence |

## Principle

Filters read surface words, not intent. A prompt about a cooking show that
mentions "knife" and "blood" (tomato juice) may trigger falsely. Rewrite
the surface while keeping the production intent intact.
