# Camera Workflow

Use this when producing a storyboard, shot list, or image/video prompt.

## Inputs To Ask For If Missing

- Scene goal
- Subject and action
- Emotion
- Power relationship
- Location and time
- Previous/next shot if continuity matters
- Output target: image prompt, video prompt, storyboard, or critique

## Shot Design Steps

1. Name the story beat in plain language.
2. Decide what the audience should know or feel.
3. Pick viewpoint.
4. Pick shot size.
5. Pick height and angle.
6. Pick composition.
7. Pick lens feeling.
8. Pick movement or stillness.
9. Add continuity anchors: screen direction, eyeline, lighting, props, emotional state.
10. Write the final output in concrete visual language.

## Single Image Prompt Format

```text
[Subject/action], [location/time], [shot size], [camera height/angle], [lens feeling], [composition], [lighting/color note], [emotion], [concrete visual details].
```

## Video Prompt Format

```text
Duration: [seconds]
Action: [one filmable beat]
Camera: [shot size, angle, movement]
Composition: [screen direction, layers, negative space]
Performance: [visible behavior]
Continuity: [props, lighting, emotion]
```
